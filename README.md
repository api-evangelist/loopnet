# LoopNet

LoopNet is the leading commercial real estate marketplace, owned by CoStar Group, providing access to property listings, sale comparables, lease rates, market analytics, and CRE investment data. The platform reaches over 13 million monthly unique visitors and covers office, retail, industrial, multifamily, land, and hospitality properties across the US, Canada, and the UK.

## API Access

LoopNet does not offer a public REST API. Integration options include:

- **LoopLink** — Embeds professional property search on third-party websites for brokers and owners.
- **Enterprise Data Access** — Programmatic access to listing data, comparables, and market analytics negotiated directly with CoStar Group for large brokerages, PropTech companies, and institutional investors.
